# Tetris game

An online tetris game built using JavaScript, HTML and CSS

## Team

- [Gloobinours](https://github.com/Gloobinours)

## How to play

Go to the link: https://gloobinours.github.io/Tetris/Front/

### Controls

- ↓ = Move faster
- → = Move right
- ← = Move left
- ↑ = Rotate
- spacebar = Snap to bottom
- c = Hold block
